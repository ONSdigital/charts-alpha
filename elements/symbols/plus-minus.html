Context

How can we make sure mathematical symbols are shown to all users?

We should probably use the minus symbol instead of a hyphen. It's also wider, so potentially more readable.

GDS recommend using the minus symbol.

Options

Option 1: plain language

positive 7 negative 3

or

plus 7 minus 3

Option 2: plus and hyphen symbols

+7

-3

-

JAWS 16 reads out "minus" only when the hyphen is next to a number

JAWS 16 reads out "dash" when the hyphen is by itself

VoiceOver for Mac skips a hyphen by itself by default

VoiceOver for iOS reads "minus" only when the hyphen is next to a number

VoiceOver for iOS reads out "hyphen" when the hyphen is by itself

NVDA reads out "minus" only when the hyphen is next to a number

NVDA skips a hyphen by itself by default

−

VoiceOver for Mac reads "minus"

VoiceOver for iOS reads "minus"

NVDA reads "minus"
