Context

The ONS website has no formal extended colour pallete. We need to introduce one so that charts and data visualisations can become more consistent.

We're also considering whether colours should be consistent with Survey data collection.

Resources

  • ONS website production colours
  • Survey data collection colours
  • Style.ONS guidance on using colour when visualising data

Colour contrast

The colour contrast ratio for text and interactive elements should be at least 4.5:1 as recommended by the W3C.

Primary colours

Salem is used for primary actions e.g. search, primary call to action buttons. Poppy is used to highlight errors, e.g. validation errors on forms. Carrot is used for global messages e.g alpha/beta banners or messages. Text links are Matisse.
