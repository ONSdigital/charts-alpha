User need

I need to know that the data is still being developed, so that I know it should be used with caution.

Research questions

  • Which design is most effective?
  • Which content works best "Experimental, draft or data being developed"?
  • Do users want to know more information?

Options

  • Option 1 (Simple alert with link)
  • Option 2 (Alert with show/hide info box)
  • Option 3 (Banner at top of page)
  • Option 4 (Alert alternative design)
  • Option 5 (Banner underneath header)
