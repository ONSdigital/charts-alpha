User need

As a user I need to perceive and understand mathemetical symbols, so that I can understand the content and data on the ONS website.

Thoughts

Use the minus symbol instead of a hyphen? This is semanticlly correct. It's also wider, so potentially more readable?

GDS recommend using the minus symbol but they actually use the en dash (not sure if accident).

Options

Option 1: plain language

positive 7 negative 3

or

plus 7 minus 3

Option 2: plus and hyphen symbols

+7 -3

Option 3: plus and minus symbols

+7 −3

Option 3: HTML abbr tag (abbreviation)

+7

-3

Option 4: aria label

+7

-3

Option 5: visually hidden class

plus+7

minus-3

minus−3

Option 6: visually hidden and aria hidden

+plus7

-minus3

Option 7: plus symbol with no mark-up, minus symbol as en dash with visually hidden "minus" text

+7

minus–3
