Important changes and notes

  • File name should probably be lower case with hyphens
  • Filenames can be too long for Windows
  • Changed column ordering to dimension1, dimension2, dimension3, value
  • Moved dimension names to column headings
  • You can't add source links (in a standard way) need to offer seperate meta data download

Other notes

  • Sensible file names (for example name, timestamp or other version indicator)
  • Avoid blank cells
  • Avoid hidden rows/columns/sheets
  • Do not merge cells
  • No labels split over multiple cells
  • Avoid including only partial time series - if so explain why and where to find
  • No evidence of overwhelming desire to print, so donâ€™t design sheets around printing
  • Don't protect workbooks
  • Geographical codes presented for each row
