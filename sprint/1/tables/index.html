Notes

Future

  • More design decisions need to be made around the preview and how it's presented and what is presented.
  • Affordance needs to be provided to the user to indicate that this is a preview.
  • Affordance needs to be provided to the user to indicate that there might be more content available via a scroll/other mechanism.
  • The full dataset view may need processing/size consideration in it's design i.e. pagination to break up the output of large datasets.
  • The full dataset link/button may need changes to emphasise/clarify it's availablity and use to the user.
  • More work needs to be done to optimise for mobile.
